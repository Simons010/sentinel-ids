## 2024-03-05 - Add Authentication to Django Rest Framework Endpoints\n**Vulnerability:** Django API endpoints exposed sensitive system configurations, reports, and network logs publicly because `IsAuthenticated` was missing.\n**Learning:** By default, if DRF `DEFAULT_PERMISSION_CLASSES` is not set globally to `IsAuthenticated`, all custom `APIView`s are public unless explicitly restricted.\n**Prevention:** Ensure `permission_classes = [IsAuthenticated]` is strictly enforced on all views or set `DEFAULT_PERMISSION_CLASSES` in `settings.py`.

## 2026-04-19 - TopBar Interactive Elements Accessibility Improvements
**Learning:** For dynamic badge counters (e.g. "7" notifications) attached to interactive buttons, assigning `aria-hidden="true"` to SVG elements and pairing them with visually hidden `<span className="sr-only">Notifications, 7 unread</span>` is a robust alternative to hardcoding `aria-label`. This approach ensures screen reader consistency without requiring programmatic updates to `aria-label` when the dynamic value changes.
**Action:** Always prefer `sr-only` siblings over `aria-label` when the associated button contains dynamic data that might get out of sync with an initial hardcoded `aria-label`. Also consistently pair focus-visible borders (`focus-visible:ring-2`) to keep keyboard navigation clear across all icon buttons.

## 2026-03-05 - TopBar Accessibility Enhancements\n**Learning:** The TopBar components rely heavily on `lucide-react` icons inside button wrappers, which previously lacked ARIA labels, creating empty stops for screen readers. Furthermore, the notification bell badge needs `aria-hidden="true"` to prevent redundant screen reader announcements when an `aria-label` is applied to the parent button.\n**Action:** Ensure all future icon-only buttons include descriptive `aria-label`s, stateful attributes like `aria-expanded` where applicable, and `aria-hidden="true"` on decorative child elements. Always include `focus-visible` styles for keyboard navigation.

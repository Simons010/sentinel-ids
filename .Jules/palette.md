## 2026-04-11 - [Frontend TopBar Accessibility]\n**Learning:** The TopBar component heavily relies on SVG icons (lucide-react) within buttons that originally lacked context for screen readers and did not have explicit focus-visible states for keyboard navigation. This meant users relying on keyboard tabbing wouldn't visually know which control was active. Incorporating Tailwind's 'focus-visible:ring-2 focus-visible:outline-none focus-visible:ring-[#22D3EE]' works perfectly within the dark-mode Sentinel theme to indicate focus state without disrupting mouse click styling. Setting aria-hidden='true' on the SVG elements and adding aria-label attributes to the wrapping elements ensures a clean, informative readout for screen readers.\n**Action:** When adding or updating interactive icon buttons across the Sentinel-IDS frontend, immediately add standard 'focus-visible' utility classes and ensure SVGs are explicitly hidden from screen readers via aria-hidden, while providing the context on the interactive parent container.
